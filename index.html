Developers' Society of BITS Goa.

DevSoc; BITS Goa

Developers' Society of BITS Goa has been primarily responsible for providing innovative technical solutions to most of the issues in BITS Goa. We began as Mobile Applications Club with the sole purpose of developing mobile applications. But after so many years we have now evolved into a club that has proficient developers from the field of Android Development, Web Development, Game Development and Internet of Things (IoT). All the four verticals work in harmony to develop innovative softwares that solve many technical issues on campus.

Portfolio Generator

Portfolio generator is our own e-portfolio creation service. Here, you can create professional portfolios for presenting while applying for various things like thesis, internships or jobs. These e-portfolios are hosted on DevSoc's efficient servers and created using the "bp-gc" keyword link.

DoJMA

We collaborated with Department of Journalism and Media for this app. This app grants the user access to BITS Herald which is the flagship publication of DoJMA and the Campus Watch that keeps the user abreast with the events on campus. It also contains contact details of taxi drivers, menus of eateries on the campus and also the mess menus. Along with this, the app also has links to important BITS sites such as moodle/photon etc.

ShortMe

This is a URL shortener service that shortens any URL you enter. The shortened URL is given a unique code with keyword "bp-gc". You can choose between a random or a custom URL and make your URL short and recognizable along with giving it a professional touch.
